# 🚀 Zaldicode | Diseño Web & Soluciones Digitales

Este es el repositorio oficial de **Zaldicode**, mi marca personal de desarrollo de software y diseño web profesional.

## 🎯 Objetivo del Proyecto

El objetivo de esta web es ofrecer soluciones digitales rápidas, modernas y optimizadas para la conversión. Aplicando una metodología basada en el **rigor científico** y la optimización técnica para ayudar a negocios a digitalizarse con éxito.

## 🛠️ Tecnologías utilizadas

Para garantizar un rendimiento de 100/100 y una carga ultrarrápida, he utilizado:

- **HTML5** Semántico para un mejor SEO.
- **Tailwind CSS** para un diseño moderno, responsivo y "Brutalista".
- **JavaScript (Vanilla)** para interactividad ligera sin sobrecargar el sitio.
- **EmailJS** para la gestión eficiente del formulario de contacto.
- **Google Fonts (DM Sans)** para una legibilidad premium.

## 📋 Características principales

- ✅ **Performance de 100/100:** Optimizada para Core Web Vitals.
- ✅ **Mobile First:** Diseño totalmente adaptado a dispositivos móviles.
- ✅ **SEO Ready:** Estructura preparada para posicionar en buscadores.
- ✅ **Legal Compliance:** Cumple al 100% con el RGPD y LSSI (España).
